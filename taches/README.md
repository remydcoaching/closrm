# Dossier Tâches — ClosRM

Ce dossier contient un fichier par tâche développée sur le projet.

## Convention de nommage

```
tache-[N]-[description-courte-en-kebab-case].md
```

Exemples :
- `tache-001-setup-initial.md`
- `tache-002-layout-sidebar.md`
- `tache-003-auth-login-register.md`

## Règles

- La numérotation est **séquentielle et globale** (partagée entre Rémy et Pierre)
- Un fichier de tâche est créé **avant ou au début** de la tâche
- Il est mis à jour **à la fin** de la tâche (statut, fichiers réellement modifiés)
- **Ne jamais supprimer** un fichier de tâche (archive historique)
