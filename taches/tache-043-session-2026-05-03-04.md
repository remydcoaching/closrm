## Apprentissages clé

- **`window.location.origin` > `NEXT_PUBLIC_APP_URL`** pour les URLs publiques générées côté browser (suit le domaine canonique de l'utilisateur)
- **Server Components > Client Components fetchant via API** quand le data est statique au moment du render (pas de flash, meilleur SEO)
- **Funnel cohorte ≠ activité globale** : ne JAMAIS afficher des chiffres décorrélés sur un funnel (ex 224 bookés vs 3 leads = absurde). Toujours partir d'une cohorte (leads créés en période) et compter ce qui leur est arrivé.
- **3 routes booking duplicate logic** (`/api/bookings`, `/api/public/book/...`, `/api/public/booking/...`) — quand on fix une, vérifier les 3.
- **`isUtilityPublicRoute` middleware Supabase** : ne pas oublier d'y ajouter les routes publiques sinon redirection vers `/dashboard` pour les users logués.
